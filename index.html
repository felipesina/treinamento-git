Quais o comandos para configurar usuário e email no git?

git config -- globaluser.name "seu nome"

git config --global user.email "seu email"

QUal a diferença entre git e github:

git é o terminal onde vamos enviar os repositorios para o github que os armazena online

O que faz o comando commit?

envia e guarda códigos em um banco de dado que no caso é o github

O que faz o comando push?

envia os arquivos para o repositório

O que faz o comando pull?

usa para baixar e armazenar repositorios do github na máquina
